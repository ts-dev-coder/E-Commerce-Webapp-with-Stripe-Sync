# 📌 Overview

このアプリケーションは、物理商品およびデジタル商品の販売を想定したE-Commerce Webアプリです。  
Stripeを用いた決済機能を備えており、カート追加や商品購入といった一連のフローを実装しています。

本アプリは、[Recursion](https://recursionist.io/) の「Backend Project 5: Server with Database」コースの課題として開発されました。
